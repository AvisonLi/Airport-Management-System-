# Project Setup and Running on Localhost


## Preparation 

The way to make this system work on localhost:

1. You ensure that Node.js has been installed on your computer 
. If not, please download it from the website of Node.js 

2. Ensure npm can be run on your computer terminal 

## Run

Suppose you require upper request, then it can run on your computer like below :

PS C:\> npm start

> node server.js
